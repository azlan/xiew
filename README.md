# xiew

Executable File View

# Build
## Requirements
* Visual C++ 2015
* Qt 5.7 - https://www.qt.io/download-open-source/

## How to build
* Pull capstone submodule
** git pull --recurse-submodules
** Build capstone\msvc\capstone_dll\capstone_dll.vcxproj
* Open xiew.pro with Qt Creator
* Press Build Project
